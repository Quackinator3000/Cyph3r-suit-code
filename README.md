# ProtoHead

A 'simple' Protogen head controller firmware.

## 

## Hardware

* Head;  -----------------------------------  Pico 2W
* Panels;  ---------------------------------  MAX7219 LED matrices (x12)
* Face controls;  --------------------------  Raspberry Pi Pico W
* WIP BATTERY SOLUTION TBD

## 

## Features

* Per eye/mouth render control
* Eye blink animation
* Quick disconnect capable wired connection (WIP WIRELESS)
* WIP audio-reactive modes
